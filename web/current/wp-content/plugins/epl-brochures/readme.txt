== Description ==

Create printable brochures of your listings easily with an added print brochure button.

== Installation ==

1. Use the plugin uploader to install the plugin Upload plugin zip contents to wp-contents/plugin/ directory and activate the plugin.


== Change log ==

= 1.3.1, December 9, 2016 =

* Tweak: Prevented Settings from being deleted unless selected in EPL settings.

= 1.3, December 8, 2016 =

* New: Ability to load custom CSS from your_theme/easypropertylistings/css/ folder.
* New: Added Google Maps API key which is pulled from the Easy Property Listings Settings.
* New: Updated included CSS from 3.1 and maintained backward compatibility with 3.0.
* Tweak: Compatibility for EPL 3.0 and 3.1 with new templates.

= 1.2, August 27, 2015 =

* New: Brochure row single template.
* New: Able to set a custom Print Brochure label on single listing button.
* New: Install function to set default values.
* New: Office details and disclaimer support some html tags, img, p, a, strong, em,  br and hr.
* Tweak: Center author details only on default brochure style.
* Fix: Correct content displayed on brochure wide when using the Staff Directory extension.

= 1.1.2, August 12, 2015 =

* Fix: Icons for bedrooms, bathrooms, parking effecting normal EPL icons.
* Tweak: CSS min height set for card template.

= 1.1.1, August 7, 2015 =

* Fix: License key not saving. Manual upgrade of this extension is required as we have corrected the error which prevented the license key from saving.

= 1.1, August 7, 2015 =

* New: Adjusted CSS for better display on some themes.
* New: Shortcode [epl_brochure] to output a single brochure by post id or property unique id.
* New: Shortcode [epl_brochures_list] to output a stock sheet/list of listings and select a default template from the extension settings.
* New: Ability to load your theme stylesheet on the brochures.
* New: Ability to load custom CSS from themes/your_theme/easypropertylistings/css/
* Tweak: Adjusted brochure buttons and brochures to noindex no follow to prevent duplicate content.

= 1.0.2, July 27, 2015 =

* Fix: Corrected image output on default brochure style.
* Fix: Removed testimonials from outputting on brochure.
* Fix: Corrected staff directory extension author.
* New: Added custom template loader to load template from theme.

= 1.0.1, May 15, 2015 =

* Fix: Corrected image output on default brochure style.
