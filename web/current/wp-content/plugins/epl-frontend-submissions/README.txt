== Description ==

Have users submit listings via the front end and manage their submitted listings.

== Installation ==

1. Upload plugin zip contents to wp-contents/plugin/ directory and activate the plugin.
2. Go to Easy Property Listings > Settings and adjust the extension settings press save.
3. Insert the [epl_frontend_submission] shortcode to add the form to a page.
3. Insert the [epl_frontend_user_listings] shortcode to a page for a user to maintain their submitted listings and edit them.
