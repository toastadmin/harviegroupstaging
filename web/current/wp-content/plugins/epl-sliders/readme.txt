== Description ==

Convert the featured image for each listing into a dynamic slider with hundreds of customisable display options.

== Installation ==

1. Install the plugin from Dashboard > Plugins > Add New > Upload. Select the downloaded plugin and activate.
2. Add your Sliders license key to Dashboard > Easy Property Listings > Extensions. This will allow updating the extension as long as your license is active.
